This is for the SENG474 ML Project

RUN CODE WITH: python3 main.py

You will be prompted to choose whether to use box art or quantitive attributes to get a rating:
"Would you like to predict a boardgame based on it's (1) box art or (2) quantitive attributes?"
- Type "1" for box art.
- Type "2" for quantitive attributes.

If you type "2", you will be prompted to choose a model:
"Would you like to run (1) random forest or (2) MLP?"
- Type "1" for random forest model.
- Type "2" for the MLP model.
